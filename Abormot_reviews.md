# Роли «Abormot reviews»
```yaml
Администратор: !role
  мотивация:
    - Подключать новых партнеров к системе
    - Выдавать доступы пользователям и операторам  
  Старт: Стартовая страница системы

Оператор: !role
  мотивация:
    - Отвечать на отзывы к товарам партнеров
    - Отслеживать новые отзывы
  Старт: Стартовая страница системы

Сотрудник партнера: !role
  мотивация:
    - Осуществлять качественную и быструю поддержку покупателям товаров
    - Отслеживать статистику по оценкам на товары
    - Редактировать ответы на отзывы
  Старт: Стартовая страница системы
 ```
# Концепция
```yaml
Отзыв: !concept
  определение: Отзыв на товар на маркетплейсе. Информация об отзыве получается из системы маркетплейса
  Обработанный отзыв: !example |
   Отзыв на который уже есть ответ.
  Необработанный отзыв: !example |
    Отзыв, на который нет ответа.
  Положительный: !example |
    Отзыв с оценкой более 3х баллов.
  Негативный: !example |
   Отзыв с оценкой менее или равный 3м баллам.

Ответ: !concept
  определение: Ответ на отзыв
  Написанный оператором: !example |
    Написан оператором врцчную.
  Сгенерированный ИИ: !extension |
    Сгенерирован на основе других ответов с помощью ChatGPT.
  Модерированный: !example |
    Ответ отредактирован сотрудником партнера.
 ```
# User story
```yaml
Администратор — Подключать новых партнеров к системе: !func
  Выдать пробный период партнеру: !story
    сделано: no
    ситуация:
      - Партнер оставил заявку на лендинге с выбором бесплатного тарифа
      - В системе создан Тариф "Пробный" с периодом действия не более месяца
      - Данные о партнере заведены в системе и выбран Тариф "Пробный"
    старт: Перехожу на главную страницу системы
    успех: 
      - Вижу близкую к 100% готовность по каждой теме по профессии
```
